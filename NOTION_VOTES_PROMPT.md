# Notion KI Prompt: Votes Database

Erstelle eine neue Database mit dem Titel "THE FORGE - Votes" mit folgender Struktur:

**Properties:**
1. **Name** (Title)
2. **Description** (Text)
3. **Votes** (Number)
4. **Status** (Select):
   - active
   - closed
   - winner
5. **Metrics** (Text) - je Zeile "Label: Value"
6. **Highlights** (Text) - eine Zeile pro Punkt
7. **Timeline** (Text) - eine Zeile pro Punkt
8. **Start Date** (Date)
9. **End Date** (Date)
